# EGX Riders — Política de Privacidad

Página estática con la Política de Privacidad de la app móvil **EGX Riders** (operada por elgenioX).

## Despliegue en GitHub Pages

1. Crear un repositorio público en GitHub (por ejemplo `riders-egx-privacy`).
2. Subir estos archivos al branch `main`:
   - `index.html`
   - `styles.css`
3. En el repositorio: **Settings → Pages → Source: Deploy from a branch → Branch: `main` / `root`** → Save.
4. La página quedará disponible en `https://<usuario>.github.io/riders-egx-privacy/`.

## Previsualización local

python3 -m http.server 8000
